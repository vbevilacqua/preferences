# Preferences



## Usage

With [docker](https://docker.com/) installed, run to build:

    $ docker-compose up

## Running swagger

We are using Swagger for easier documentation and to allow us to call the api easier.

Access the webpage from any browser:

```
    https://localhost:7122/swagger/
```
